How to use Chromium to Learn Code

Chrome OS, Google Chrome, Microsoft Egde, Opera and Opera GX
Ariel Araya, July 11, 2020

Downloading Learn Code is as easy as going to the Learn Code website and clicking download. Chromium is the open source project between Google Chrome, Chrome OS, and Microsoft Edge (Support for Opera coming soon). To get started go to your browsers extentions page by pressing the three dot up in the top right corner and then go to the bottom left corner and click on the "Enable Developer Mode" setting to turn it on. After you turn Developer Mode on. Find "Lesson 1" and you just started your coding journey. Good luck!

Note: This program group assumes that you are using Chrome OS. If you are not, the intstructions are quite the same.
