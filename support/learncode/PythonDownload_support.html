How to use Python to Learn Code

Windows OS, MacOS Mavericks to Big Sur, and Linux
Ariel Araya, July 9, 2020

Downloading Learn Code is as easy as going to the main learn code website, and clicking Download Python. Make sure that you have IDLE installed on your computer. You can do that from going to python.org and on the menu pressing "Download". When you have everything done, extract the folder and Double click on Lesson 1 - HTML. Congrats! You have just started your coding journey.
